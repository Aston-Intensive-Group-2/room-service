## README

для запуска всех сервисов вводим:

```bash
    mvn clean install
    docker compose up
```
___
### Запустить отдельно конкретный микросервис:
Все сервисы работают со своей БД, поэтому перед запуском сервиса запускаем контейнер с БД,
а так же сервис api-gateway и discovery-server:
* поднимаем БД: docker compose up <имя сервиса>, например
```bash
    docker compose up api-gateway, discovery-server, users-service-db, users-service
```
* заходим в папку микросервиса и стартуем его. Для фанатов консоли:
```bash
    mvn spring-boot:run
```

### api-gateway
Это шлюз нашего приложения. Крутится по адресу http://localhost:8080
и перенаправляет запросы на наши микросервисы - это основная точка входа в наше приложение.

### discovery-server
Регистрирует наши микросервисы для gateway. Крутится по адресу http://localhost:8761
___

http://localhost:8080/api/v1/admins<br>
http://localhost:8091/api/v1/admins<br>
Административный контролер
<hr/>

http://localhost:8080/account/login<br>
http://localhost:8091/account/login<br>
Точка входа, получение JWT-токена по мылу и паролю
<hr/>

Сервис для менеджмента пользователей. Отвечает за:
* создание пользователя
* редактирование и удаление пользователя
* выдачу и валидацию JWT-токенов
* выдачу DTO пользователя по валидному JWT

### booking-service:
http://localhost:8080/api/v1/bookings<br>
http://localhost:8092/api/v1/bookings<br>
Сервис для создания и отправки уведомлений. Отвечает за:
* бронирование комнаты
* отмену бронирования
* проверку коллизий бронирования по времени
* отображение свободных для бронирования дат
* историю бронирования

### room-service:
http://localhost:8080/api/v1/rooms<br>
http://localhost:8093/api/v1/rooms<br>
Сервис для создания и отправки уведомлений. Отвечает за:
* отображение списка помещений
* создание, редактирование, удаление помещений (с правами администратора)
* фильтрация по параметрам (размерам)
* оснащение?

### notifications:
http://localhost:8080/api/v1/notifications<br>
http://localhost:8094/api/v1/notifications<br>
Сервис для создания и отправки уведомлений. Отвечает за:
* создание уведомлений
